#This prtoject is a Blogging Web App which is developed using Asp.net core 8.0 with Sql Server 2022 and Entity Framework
#IDE used VS 2022
#After downloading/Cloning the project Configure your appsettings.json file 

1. Install sql server 2022 and ssms (ss management studio). Then create your username, pwd. Ater that, Create a Database name in your sql server. Now

2. Configure your appsettings.json file using your own db name,pwd. For EX: "DefaultConnection": "Server=localhost; Database=your db name; User Id=sa; Password=your pwd; TrustServerCertificate=True"
